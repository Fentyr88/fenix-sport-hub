# Welcome to your project

Requisitos
Node.js (recomendado: 18+; si tienes 20+ mejor)
npm (viene con Node) o pnpm/yarn (opcional)

Verifica versiones:
node -v
npm -v

Instalación:
Clona o descarga el proyecto y entra a la carpeta:
cd fenix-sport-hub

Instala dependencias:
npm install


Ejecutar en desarrollo (modo dev)
npm run dev

Luego abre en el navegador:
http://localhost:8080
